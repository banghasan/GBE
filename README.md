# telegram-bot-api-engine
bangHasan Bot Engine for Telegram Bot API

## Menjalankan

### 1. Download
Download release terbaru sesuai OS kalian masing-masing.

### 2. Edit .env

- Buka dan edit file `.env`
- Sesuikan isi di dalamnya, terutama Token Bot API nya

### 3. Jalankan

#### Linux

```bash
$ ./bHBE_x64 
```

#### Windows

```cmd
bHBE.exe
```

### 4. Selesai

Test Bot nya!
